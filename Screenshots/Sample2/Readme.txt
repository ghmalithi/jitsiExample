2) Sample2 : To get a full screenshot of a specific window by its ID.
--------------------------------------------------------------------------------

Class Name: src/jniex/samples/Sample2.java

1) To run the sample, cd to ./distribution folder and execute

java -Djava.library.path="../lib/native/linux-64" -jar JitsiScreenCapture.jar

2) Select the sample number as 2. 

3) It will show a list of available windows. Enter the id of the window you want to get the screen shot.

4) Then it will take a full screenshot of the window specified by id. If printscreen is success, the image will be automatically previewed.

5) The image is scrollable. See the full image by scrolling.
